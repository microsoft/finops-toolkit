## Problem
During evaluation of `src/templates/sre-agent`, a broad glob over `src/templates/sre-agent/**` timed out and saved partial output to a temp file. A subsequent `**` glob returned 2.2 MB of output and was too large to read at once.

## Expected behavior
Template evaluation workflows should use bounded file discovery patterns or tooling should expose a cheap directory summary mode for large template trees.
